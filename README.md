# ガス主任Pass プライバシーポリシー

最終更新日: 2026年6月28日

ガス主任Pass（以下「本アプリ」）は、ガス主任技術者試験の学習を支援する非公式アプリです。本ポリシーは、本アプリにおける利用者情報の取り扱いについて説明するものです。

本アプリは、一般財団法人 日本ガス機器検査協会、経済産業省、その他の公式機関が提供、承認、監修、提携するアプリではありません。

## 1. 本アプリが取り扱う情報

### 1.1 端末内に保存される学習データ

本アプリは、学習機能を提供するため、以下の情報を利用者の端末内に保存します。

- 回答履歴
- 正誤履歴
- 正答数、回答数、正解率などの集計情報
- 分野別の学習進捗
- 誤答優先学習に必要な正答数、誤答数
- 苦手問題、ブックマークの情報
- 連続学習日数
- 甲種、乙種、丙種の対象種別設定
- 文字サイズなどのアプリ設定

これらの情報は、問題演習、復習、学習状況の表示、誤答優先学習、苦手問題機能、実戦演習、アプリ設定の保持のために使用します。

現時点で、本アプリ独自のアカウント登録機能、ログイン機能、学習履歴のサーバー同期機能はありません。

### 1.2 Firebase Analytics に関する情報

本アプリは、利用状況の把握と品質改善のために Firebase Analytics を利用します。Firebase は Analytics only として利用します。

送信するイベントは、アプリ起動、初回設定完了、学習開始、回答、学習完了、購入画面表示、購入開始、購入完了、レビュー依頼条件到達など、アプリの利用状況を集計するための標準イベントに限定します。

Firebase Analytics へ以下の情報は送信しません。

- 問題本文
- 選択肢本文
- 解説本文
- question_id の生値
- 氏名、住所、電話番号、メールアドレス
- 問い合わせ本文
- 購入トランザクションID
- 決済ID
- 広告ID
- AdMob ID
- Firebase App ID
- RevenueCat SDKキー
- その他の内部識別子や秘密情報

本アプリでは、Firebase Authentication、Firestore、Realtime Database、Cloud Functions、Storage、Remote Config、Crashlytics、Cloud Messaging、App Check は使用しません。

Firebase による情報の取り扱いについては、Google のプライバシーポリシーをご確認ください。

- Google プライバシーポリシー: https://policies.google.com/privacy

### 1.3 広告に関する情報

本アプリは、無料版の広告表示のために Google Mobile Ads SDK（AdMob）を利用する場合があります。広告配信、広告の品質管理、不正防止、効果測定のため、端末情報、広告の表示や操作に関する情報などが Google により取り扱われる場合があります。

Premium を購入した場合、アプリ内の広告表示は非表示になります。

Google による広告関連情報の取り扱いについては、以下をご確認ください。

- Google プライバシーポリシー: https://policies.google.com/privacy
- Google 広告に関するポリシー: https://policies.google.com/technologies/ads

### 1.4 アプリ内購入に関する情報

本アプリは、買い切り Premium の購入状態を管理するために RevenueCat SDK を利用します。購入状態の確認、購入の復元、Premium 機能の有効化のため、購入情報、アプリユーザーID、端末情報などが RevenueCat および Apple により取り扱われる場合があります。

購入処理は Apple の App 内課金の仕組みにより行われます。本アプリの開発者は、クレジットカード番号などの支払い手段の詳細を取得しません。

各サービスによる情報の取り扱いについては、以下をご確認ください。

- RevenueCat Privacy Policy: https://www.revenuecat.com/privacy/
- Apple Privacy Policy: https://www.apple.com/legal/privacy/

### 1.5 レビュー依頼に関する情報

本アプリは、一定の学習条件を満たした場合に、Apple が提供するレビュー依頼機能を表示する場合があります。レビューの入力、送信、表示、管理は Apple の仕組みにより行われます。本アプリの開発者が、レビュー内容と利用者個人を結び付けて取得することはありません。

### 1.6 診断・不具合調査に関する情報

本アプリの安定性向上、不具合調査、品質改善のため、OS バージョン、端末種別、アプリバージョン、クラッシュや診断に関する技術情報が Apple、Expo、または利用している開発基盤により取り扱われる場合があります。

本アプリでは Firebase Crashlytics は使用しません。

### 1.7 お問い合わせ時に利用者から提供される情報

利用者がお問い合わせを行った場合、メール本文などに記載された内容、返信先メールアドレス、端末やアプリの状態に関する情報を、問い合わせ対応のために利用する場合があります。

お問い合わせで取得した情報は、回答、調査、再発防止のために必要な範囲でのみ使用します。

## 2. 利用目的

本アプリは、取り扱う情報を以下の目的で利用します。

- 問題演習、復習、学習進捗表示の提供
- 甲種、乙種、丙種の対象種別に応じた出題
- 誤答優先学習、苦手問題、連続学習日数表示の提供
- 実戦演習と学習結果表示の提供
- 文字サイズなどの利用者設定の保存
- 広告の表示、広告品質の管理、不正防止
- 利用状況の分析と学習体験の改善
- アプリ内課金の購入状態確認、復元、Premium 機能の提供
- 不具合、表示崩れ、動作不良などの調査と改善
- 利用者からのお問い合わせへの対応

## 3. 第三者提供・外部送信

本アプリは、広告表示、利用状況分析、アプリ内購入、品質改善のため、以下の外部サービスを利用する場合があります。

- Firebase Analytics
- Google Mobile Ads SDK（AdMob）
- RevenueCat
- Apple App Store / StoreKit
- Expo および Apple が提供する診断関連機能

各サービスで取り扱われる情報は、それぞれのプライバシーポリシーに従って管理されます。

本アプリの学習履歴は、原則として利用者の端末内に保存されます。ただし、OS、SDK、外部サービスの仕様により、広告、購入、分析、診断に関する情報が各サービスへ送信される場合があります。

## 4. 収集しない情報

本アプリは、以下の情報を本アプリ独自の機能として収集しません。

- 氏名、住所、電話番号
- 連絡先
- 正確な位置情報
- 写真、カメラ、マイクの内容
- 本アプリ独自のログインID、パスワード
- 勤務先、実務経験証明、受験番号
- ガス主任技術者免状の情報
- 試験申込情報、試験結果、成績通知
- 実務相談、保安監督、設計、施工、検査、法令適合診断の依頼内容
- 学習履歴を外部サーバーへ同期するためのアカウント情報

お問い合わせを行った場合は、利用者が本文に記載した情報を問い合わせ対応のために取り扱います。

## 5. IDFA・トラッキングについて

本アプリは IDFA を使用しません。App Tracking Transparency の許可を必要とするトラッキングは行いません。

Firebase Analytics は IDFA を使用しない構成で利用します。本アプリは、第三者のアプリやWebサイトを横断して利用者を追跡する目的で情報を利用しません。

## 6. 学習データの削除

利用者は、アプリ内の設定画面から学習データをリセットできます。アプリを削除した場合も、端末内に保存された学習データは削除されます。

Apple、Google、RevenueCat など外部サービス側で管理される購入情報、広告関連情報、診断情報については、各サービスのポリシーおよび設定に従って管理されます。

## 7. 未成年の利用

本アプリは資格学習を目的としたアプリです。未成年の方が利用する場合は、必要に応じて保護者の方と一緒に内容を確認してください。

## 8. 学習コンテンツと公式機関との関係

本アプリは、ガス主任技術者試験の学習を目的とした非公式アプリです。一般財団法人 日本ガス機器検査協会、経済産業省、その他の公式機関が提供、承認、監修、提携するものではありません。

本アプリの問題および解説は、公開されている試験概要、出題範囲、法令・技術分野の一般的な学習事項を参考に、スマートフォン学習向けに独自作成しています。

公式問題、過去問題、公式解答、公式教材本文、公式図版、公式ロゴは収録していません。

本アプリは、試験の申込、受験、試験結果の確認、資格認定、免状申請、実務経験証明の作成または代行を行いません。試験制度、申込方法、受験資格、出題範囲などの最新情報は、必ず公式情報をご確認ください。

## 9. ポリシーの変更

本ポリシーは、法令、外部サービスの仕様変更、アプリ機能の変更に応じて改定される場合があります。重要な変更がある場合は、アプリ内または公開ページ上で分かりやすく告知します。
