# ガス主任Pass プライバシーポリシー

最終更新日: 2026-06-28

ガス主任Pass（以下「本アプリ」）は、ガス主任技術者試験の学習を支援する非公式アプリです。一般財団法人 日本ガス機器検査協会、経済産業省、その他の公式機関とは関係ありません。

## 取得する情報

本アプリは、学習機能、広告表示、購入機能、利用状況分析のため、以下の情報を取り扱う場合があります。

- 学習進捗
- 正答数、回答数、正解率などの集計情報
- アプリ内購入の状態
- 広告表示に必要な情報
- アプリの利用状況に関する集計情報

## Firebase Analytics

本アプリはFirebase Analyticsを利用します。利用目的は、アプリ起動、初回設定完了、学習開始・完了、購入開始・完了などの集計分析です。

以下の情報はFirebase Analyticsへ送信しません。

- 問題本文
- 選択肢本文
- 解説本文
- question_id生値
- 氏名、メールアドレス等の個人情報
- 問い合わせ本文
- 決済ID
- 広告ID

Firebase Auth、Firestore、Realtime Database、Cloud Functions、Storage、Remote Config、Crashlytics、Cloud Messaging、App Checkは使用しません。

## 広告

本アプリはGoogle AdMobによる広告を表示する場合があります。Premium購入後は広告が非表示になります。

## アプリ内購入

本アプリはRevenueCatを利用して、買い切りPremiumの購入状態を管理します。購入処理はAppleの仕組みを通じて行われます。

## IDFA / トラッキング

本アプリはIDFAを使用しません。App Tracking Transparencyの許可を必要とするトラッキングは行いません。

## 公式コンテンツについて

本アプリの問題・解説は独自作成です。公式問題、過去問題、公式解答、公式教材本文、公式図版、公式ロゴは収録していません。
